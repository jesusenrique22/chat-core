Simulación de Proyecto Externo

Servicios Maracaibo - Sitio Web Oficial

Este archivo simula la web del compañero en otro puerto, cargando el widget desde tu conector en la red local (sin VPN).

Uso

  • Levanta el conector: npm run backend (puerto 4000).
  • Sirve esta carpeta: npx serve demo-cliente-externo -p 8181
  • Abre con la IP de quien hostea el chat, ej:
    http://localhost:8181?server=192.168.1.50
  • server = IP LAN de la máquina donde corre el backend.
